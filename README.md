##我还可以得到Dualx的源码吗
请您放心，Dualx将永远没有任何附加条件地遵循GPL发布，您无论何时都可以自由免费地得到完整的Dualx源码，同时我们鼓励您在遵循GPL协议的条件下传播Dualx或由Dualx产生的衍生作品及其源码。只要您遵循非商业、署名和衍生作品开源的原则，Dualx可以被您自由利用。
