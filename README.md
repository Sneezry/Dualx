Dualx是一款基于Chrome的QQ客户端，您可以在使用基于版本为*25.0.1364.97*及以上的chromium内核的Chrome浏览器、Chrome OS操作系统及其他平台中使用。

Dualx使用WebQQ协议登录并与腾讯服务器进行通信，登录及通信过程只与*.qq.com域进行数据交换。

Dualx是*开源*的项目，遵循*GPL协议*发布。
